CAMPHOR- Advent Calendar

Requirements
------------
- Docker
- docker-compose

Edit
----
0. Edit [data/authors.yml](data/authors.yml)
0. Edit [data/entries.yml](data/entries.yml)

Compile & Development
---------------------
0. Run `docker-compose up -d`
0. Open http://localhost:8000

Deploy
------
0. Run `./deploy.sh` or `./deploy.sh <branch>`
